# Extra Ruleset

自动从 v2ray-rules-dat 提取特定规则，生成代理规则列表。

## 生成的文件

- **AI DOMAIN-SET** - AI 相关服务代理规则（排除中国）
- **direct-game DOMAIN-SET** - 游戏相关服务代理规则（中国区）
